## üîç How AI Agents Use This Knowledge

### 1. Policy Agent
- Extracts policy parameters from user input
- Retrieves related policies from knowledge base
- Provides context about existing similar policies
- Suggests policy modifications based on successful implementations

### 2. Impact Agent
- Uses historical policy data to predict impacts
- Compares with similar policies in other states
- Provides evidence-based impact estimates
- References actual policy outcomes

### 3. Behavior Agent
- Analyzes citizen response to similar policies
- Uses data from implemented schemes
- Predicts adoption rates based on historical data
- Considers cultural and regional factors

### 4. Optimization Agent
- Suggests improvements based on successful policies
- Recommends budget allocations from similar schemes
- Identifies best practices from other states
- Proposes implementation strategies

### 5. Explainability Agent
- References actual policies to explain predictions
- Provides examples from real implementations
- Cites government sources and data
- Makes recommendations transparent and credible

### 6. Simulation Agent
- Uses real policy parameters for simulations
- Incorporates actual budget allocations
- Models based on historical policy impacts
- Provides realistic scenario analysis
